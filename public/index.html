price-engine

API that aggregates secondhand market prices from Vinted, eBay, and Vestiaire Collective.

Endpoint

POST /.netlify/functions/scrape-prices

Request body

{
  "brand": "Hermès",
  "category": "Taschen",
  "productName": "Birkin 30",
  "bypassCache": false
}

Response shape

{
  "vinted":    { "listings": [...], "pagesScraped": 3 },
  "ebay":      { "listings": [...], "pagesScraped": 3 },
  "vestiaire": { "listings": [...], "pagesScraped": 1 },
  "meta": {
    "counts":     { "vinted": 73, "ebay": 121, "vestiaire": 15 },
    "durationMs": 14563,
    "cached":     false,
    "diagnostics": ["vinted home: HTTP 200", ...]
  }
}
